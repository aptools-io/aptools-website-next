# APTOOLS.IO

Requirements:

```
node -v >= 16
```

Installing:

```
git clone git@github.com:aptools-io/aptools-website-next.git in main projects folders
npm i
```

Production deployment:

```
npm run build
npm run start
```

Dev deployment with watcher:

```
npm run dev
```

Default deployment port - 3000
