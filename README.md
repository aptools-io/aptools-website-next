# APTOOLS.IO

Requirements:
node -v >= 16

Installing:

```
git clone git@github.com:aptools-io/aptools-website-next.git in main projects folders
npm i
```

Deployment:

```
npm run build
npm run start
```

or deployment with watcher:

```
npm run dev
```
